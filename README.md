# EC2 Static Website Hosting using Nginx

This project demonstrates how to host a static website on an AWS EC2 instance using the Nginx web server.

## Services Used
- AWS EC2
- Amazon Linux 2
- Nginx
- SSH

## Project Steps
1. Launched an EC2 instance (Amazon Linux 2)
2. Configured security groups (SSH & HTTP)
3. Installed and configured Nginx
4. Deployed a static HTML website
5. Accessed the website via EC2 public IP

## Output
The website is accessible via the EC2 public IPv4 address.

## Note
